# KC Streetcar Sunday, July 17 Service

Source: https://kcstreetcar.org/kc-streetcar-sunday-service/

For this Sunday ONLY , July 17, KC Streetcar service will begin at 6:00am as opposed to 7:00am, which is normal operations. This is to accommodate anticipated crowds for a YMCA 5K planned for the Crown Center area.
