# Detours Ahead!

Source: https://kcstreetcar.org/detours-ahead/

## 3rd & Grand Track Work to start June 15.

Work on one of the last pieces of track will start the week of June 15, 2015. This is the special turn piece that will connect Grand to 3rd St. During work, 3rd will be closed from Walnut-Grand and Grand will be open to southbound only from 3rd to Missouri Ave. This will be in place for 3 weeks, including weekends.
