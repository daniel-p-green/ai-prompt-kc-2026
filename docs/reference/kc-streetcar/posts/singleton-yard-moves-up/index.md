# Singleton Yard Moves Up

Source: https://kcstreetcar.org/singleton-yard-moves-up/

## Pre-cast walls to be installed using 133-foot tall crane

The KC Streetcar Constructors marks another milestone as walls go up on the Singleton Yard vehicle maintenance facility. Holmes between 1st and 3rd Streets will be closed to thru traffic for one week during this work.
