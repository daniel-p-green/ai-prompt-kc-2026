## Slide 4: 60-Day Pilot Operating Model

**Judge question:** How does this work in practice?

**Title:** Start Narrow. Prove Fast.

**Answer:**  
StreetCart KC runs recurring hub windows, bilingual resident guidance, and selective delivery support through a simple operating loop.

**Operating loop:**
1. Supply and partner window confirmed
2. Hub goes live
3. Residents get bilingual instructions
4. Pickup or delivery gets verified
5. Public scorecard updates

**Visual:**  
One operating loop plus a short `stand up / run / measure` timeline
