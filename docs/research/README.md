# Research

This folder is for deep research artifacts that inform the Kansas City food access work.

## Structure

- One subfolder per research run
- `prompt.md` stores the exact source prompt
- `output.md` stores the returned output

## Naming

Use date-first folder names so entries sort cleanly:

- `YYYY-MM-DD-source-topic`

Example:

- `2026-03-28-perplexity-city-food-access-programs`

## Notes

- Keep raw research outputs intact unless there is a clear cleanup reason.
- Add synthesized takeaways in product docs separately; this folder is the source record.
